## Code Blocks

Add a language after the opening fence when you can:

```bash title="terminal"
ros2 pkg create waypoint_robot --build-type ament_python
cd waypoint_robot
```

```cpp title="motor-test.cpp"
void loop() {
  setMotorPower(leftMotor, 120);
```
