# Waypoint

Website still a Work in progress! Come back in a week or so!

## Overview

Waypoint is a You Ship, We Ship (YSWS) program where you design your own ROS2 robot.

### You ship:

- A PCB design for your robot
- A chassis design in .STEP format
- A starter ROS2 package for the robot

### We send you:

- A custom Hack Club Waypoint kit!
- A $15 card grant to get your PCB ordered from JLCPCB, and a 3D printed case from [printing legion](https://printlegion.hackclub.com/)!

## Development

The website lives in [`website/`](website).

```bash
cd website
npm install
npm run dev
```
